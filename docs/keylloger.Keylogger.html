keylloger.Keylogger

class documentation

class Keylogger:

Constructor: Keylogger(interval, report_method)

View In Hierarchy

Класс Keylogger реализует функционал для записи нажатий клавиш, их обработки и отправки отчетов.

Методы: - prepare_mail: подготавливает письмо для отправки с помощью SMTP. - sendmail: отправляет подготовленное письмо через указанный SMTP-сервер. - report: собирает данные, отправляет отчет и запускает следующий таймер. - start: запускает keylogger и обработчик событий клавиатуры.

Parameters
intervalинтервал времени между отправкой отчетов (в секундах)
report_methodметод отправки отчетов (по умолчанию "email")
