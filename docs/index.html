def on_release(key): ¶

Обрабатывает событие отпускания клавиши и выполняет соответствующие действия.

Функционал: - Определяет текущую раскладку клавиатуры. - Завершает ввод при нажатии Enter. - Добавляет пробел в буфер при нажатии Space. - Конвертирует символы в зависимости от текущей раскладки (RU или EN). - Удаляет последний символ из буфера при нажатии Backspace. - Удаляет клавишу из глобального списка pressed_keys. - Завершает работу программы, если нажата клавиша ESC.

Исключения: - Обрабатывает любые исключения, возникающие во время выполнения логики. - Выводит сообщение об ошибке в консоль.

Parameters
key:pynput.keyboard.Key или strклавиша, которая была отпущена
Returns
None или boolFalse, если клавиша ESC нажата для завершения работы, иначе None
def process_inputs(): ¶

Обрабатывает введенные данные и классифицирует их как логин, пароль или неизвестные данные.

Обработка: - Собирает текст из глобального буфера inputs. - Если строка определена как email, логин или номер телефона, устанавливает тип "Login". - Если предыдущий ввод был "Login", классифицирует текущий ввод как "Password". - Если строка соответствует критериям пароля, классифицирует как "Password". - В противном случае классифицирует как "Unknown".

Данные сохраняются в глобальной переменной collected_data с меткой времени и активным окном.

Returns
NoneNone
